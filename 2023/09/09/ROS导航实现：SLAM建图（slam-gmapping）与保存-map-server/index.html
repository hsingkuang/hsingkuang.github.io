ROS导航实现：SLAM建图（slam_gmapping）与保存(map_server)

先安装相关的ROS功能包:

  新建功能包，并导入依赖: gmapping map_server amcl move_base，其中gampping用于构建地图，map_server用于保存与读取地图，amcl用于定位，move_base用于路径规划

编写gmapping相关的launch文件

启动Gazebo仿真环境和编写好的slam.launch文件

在RVIZ中添加组件

添加机器人模型RobotModel
添加Laserscan并设置Topic话题scan
添加坐标系组件TF，Fram下即可找到map，odom
添加map组件用于建图，设置topic，发布话题栅格地图数据

启动键盘控制节点

保存地图map server

读取地图信息（map_server包里的map_server）

我们发布的信息以话题的形式发布，要看到地图可视化可以使用rviz，终端打开rviz并加入map插件，选择topic为map，即可看到效果。
